# Shine Painting Service

Professional website for Shine Painting Service — a 5-star rated, family-run painting business based in Preston, VIC.

## Tech Stack

- **Framework**: Next.js 14 (App Router) with TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Netlify (static export)

## Features

- Fully responsive design
- Scroll-reveal animations
- Free quote request form
- Services showcase
- Client reviews section
- Direct click-to-call CTA

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `/out` — a fully static site ready for any CDN or Netlify.
